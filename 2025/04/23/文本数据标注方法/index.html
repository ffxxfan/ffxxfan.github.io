1.2 deccano 运行与使用

按照 1.1 步骤完成后，在浏览器中打开 http://127.0.0.1:8000/ （端口号需为自己设定的端口号），可以见到如下页面：

我们可以在右上角切换「语言」和 进行「登录」，现在，让我们开始点击中间的「GET STARTED」开始吧！！！

  • 使用 1.1.3 doccano 初始化 时注册的 super user 进行登录，账号和密码分别是 「your_admin」、「your_password」；

  • 点击 「Create」创建一个新的项目，选择项目任务，填写相关内容（项目名称、描述、Tags），勾选相关权限内容（如果新手有什么不理解的直接问 deepseek 就好）；

  • 创建成功后将进入主页页面，就可以进入官方教程啦！！！此处不再赘述官方教程。

  • 然后你就可以按照官方教程进行标注并导出结果啦！！！

1.3 可能遇到的问题及解决办法

1.3.1 AttributeError: module ‘marshmallow’ has no attribute ‘version_info’

可能是 marshmallow 的版本太高导致的。

  • 我们可以在 github 中的 doccano 项目中看到，上次版本更新为 2023/06/14；
  • 通过 pip list 查看已安装的 marshmallow 版本（我的是 4.0.0）；
  • 然后我们可以在 PyPI 中查看 marshmallow 版本，发现 marshmallow 4.0.0 版本更新于 2025/04/17，已经和 marshmallow 发布时间差了两年了！！！
  • 所以我们尝试回退 marshmallow 版本，我们可以看到，2023 marshmallow 版本为 3.20 版本，于是选择回退该版本，我选择的是 3.20.2（成功啦），也可以尝试别的版本√；

可能是 marshmallow 的版本太高导致，回退版本 3.20.2（也可以选择其他的）：
