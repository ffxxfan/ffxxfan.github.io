hexo 常用命令及 Bug 解决

0x00 前言

本文记录了在 hexo 的常用命令及在使用过程中遇到的 Bug。

0x01 hexo 常用命令

命令 说明
hexo init <folder> 初始化 Hexo 项目（新建站点）
hexo clean 清除缓存和生成的 public 文件夹
hexo generate 或 hexo g 生成静态文件（到 public 目录）
hexo server 或 hexo s 启动本地服务器（默认 http://localhost:4000）
hexo deploy 或 hexo d 部署到远程（如 GitHub Pages）
hexo new <title> 新建一篇博客文章（在 source/_posts 目录）
hexo new page <pageName> 新建一个页面（如 About 页）
hexo list <type> 列出文章/页面等（如 hexo list post）

hexo 官方链接：https://hexo.io/zh-cn/

0x0201 MathJax 渲染

Bug 向 - Hexo - MathJax 数学公式渲染问题； Hexo 部分 MathJax 渲染能够成功，部分渲染失败，涉及部分转义词如”_{“、“}{”等会自动进行转义，进而无法正常渲染公式。
