计算机相关的专有名词解释

一、计算机网络相关

DNS: Domain Name System，是域名系统。DNS 就是把域名解析为 IP 地址，提供我们上网，我们能够上网最终是找到 IP 地址。比如，https://ffxxfan.github.io/(ffxxfan.github.io) 是域名，那么它的 IP 地址假设是 202.117.80.6，我们用浏览器访问该的网页，先到 DNS 服务器找 https://ffxxfan.github.io/ 对应的 IP 地址，然后通过 IP 地址进行访问。

DNS 污染：DNS cache pollution。又称域名服务器缓存投毒（DNS cache poisoning），是指一些刻意制造或无意中制造出来的域名服务器（DNS）数据包，把域名指往不正确的IP地址。一般来说，在互联网上都有可信赖的网域服务器，但为减低网络上的流量压力，一般的域名服务器都会把从上游的域名服务器获得的解析记录暂存起来，待下次有其他机器要求解析域名时，可以立即提供服务。一旦有关网域的局域域名服务器的缓存受到污染，就会把网域内的计算机导引往错误的服务器或服务器的网址。

二、计算机视觉

Prompt：是用户提供给指向模型（可以是 GPT 等语言模型 AI，也可以是 SAM 等分割模型）提供输入以引导其生成特定输出的文本或指令，用于引导模型生成特定的输出。它可以是一个问题、一段指令、一个关键词，甚至是一段不完整的文本或者图像上的点。模型根据 prompt 理解用户的意图，并生成相应的回答或内容。

也就是说，模型会根据用户给的 prompt 生成用户想要的东西。

prompt 用于提示，不局限于形式，如 图像、点（如图像上的某个像素）、文本等，只要是能够给 模型 带来 「启发式思考」的东西，都可以称作 Prompt。

生成模型：生成模型是一类能够学习数据分布并生成新数据的机器学习模型。它们可以模仿训练数据的统计特征，生成类似但全新的样本，如图像、文本、音频等。与判别模型（Discriminative Models，如分类器）不同，生成模型的目标是建模数据的生成过程，而非仅仅区分不同类别。目前主流的方法有：GAN、Diffusion Models、VAE、flow matching 等。
