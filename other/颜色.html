0x01 相关代码

0x01A 内联样式 span

功能 示例代码 效果说明
设置字体颜色 <span style="color:red">红色文字</span> 设置文字为红色
使用十六进制颜色 <span style="color:#28a745">绿色文字</span> 自定义颜色（绿色）
使用 RGB 颜色 <span style="color:rgb(0, 123, 255)">蓝色文字</span> RGB 格式设置颜色
设置字体大小 <span style="font-size:20px">20px 大小</span> 设置字体为 20px
使用 em 设置大小 <span style="font-size:1.5em">1.5em 大小</span> 相对于父元素的大小
设置加粗 <span style="font-weight:bold">加粗文字</span> 字体加粗
设置斜体 <span style="font-style:italic">斜体文字</span> 字体倾斜
多种样式组合 <span style="color:purple; font-size:18px; font-weight:bold">样式</span> 同时设置颜色、大小、粗体

0x01B 内联样式 font

[!note]

以下是 <font> 标签相关属性的表格总结（HTML 中已废弃，但部分 Markdown 编辑器或旧网页仍支持）：

推荐使用 <span style="..."> 替代 <font>，因为 <font> 是 HTML 4 的遗留标签，在 HTML5 中已被废弃，不再推荐使用。

属性名称 示例 说明
color <font color="red">红色文字</font> 设置字体颜色（可用名称或十六进制）
size <font size="4">字体大小</font> 设置字体大小（1~7，默认是3）
face <font face="Arial">Arial 字体</font> 设置字体族（如 Arial、Courier）
多属性组合 <font color="#00aa00" size="5" face="Verdana">组合</font> 同时设置颜色、大小和字体
